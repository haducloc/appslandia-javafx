## Installation

### Maven
```XML
<dependency>
    <groupId>com.appslandia</groupId>
    <artifactId>appslandia-javafx</artifactId>
    <version>{LATEST_VERSION}</version>
</dependency>

<dependency>
    <groupId>com.appslandia</groupId>
    <artifactId>appslandia-common</artifactId>
    <version>{LATEST_VERSION}</version>
</dependency>
```

### Gradle
```
dependencies {

   compile 'com.appslandia:appslandia-javafx:{LATEST_VERSION}'
}
```
